# @lbwa/prettier-config

An opinionated prettier config.

## Install

```bash
npm i @lbwa/prettier-config -D
```

## Usage

add `prettier` field in `package.json`:

```json
{
  "prettier": "@lbwa/prettier-config"
}
```
